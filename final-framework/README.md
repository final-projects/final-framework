# final framework

`final-framework`用于封装常用的技术框架。

|                         artifactId                         |                        描述                        |
| :--------------------------------------------------------: | :------------------------------------------------: |
|                        `final-core`                        |                                                    |
|                     `final-annotation`                     |     定义基本的`Annotation`，为框架解析提供基础     |
|                      `final-context`                       |                                                    |
|    [`final-json`](final-json/README.md)    |         定义统一的Json序列化与反序列化接口         |
|                       `final-redis`                        |                                                    |
|                       `final-cache`                        |               基于`AOP`的的缓存切面                |
| [`final-mybatis`](final-mybatis/README.md) |      定义通用的`Mapper`，提供强大的`CRUD`功能      |
|     [`final-web`](final-web/README.md)     |  封装`springMvc`，提供增强的拦截器、异常处理机制   |
|                        `final-aop`                         |                                                    |
|                        `final-data`                        |                                                    |
| [`final-monitor`](final-monitor/README.md) | 提供丰富的切面监控，如**操作日志**，**链路追踪**等 |
