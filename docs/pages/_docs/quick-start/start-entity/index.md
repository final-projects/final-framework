# Start Entity

## What

`final-entity` define the rule of `ORM`(**Object/Relational Mapping**).

## How

### Import Dependency

* maven

```xml

<dependencies>
    <dependency>
        <groupId>org.ifinal.finalframework.boot</groupId>
        <artifactId>final-boot-starter-entity</artifactId>
        <version>{{ site.final.version }}</version>
    </dependency>
</dependencies>
```
