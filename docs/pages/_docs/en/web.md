## 统一的结果集

你在`@RestController`中定义的方法：

```java

@RestController
@RequestMapping("/hello")
public class HelloController {

    @RequestMapping
    @MonitorAction("${'{访问Hello} ' + #word}")
    public String hello(String word) {
        return "hello " + word + "!";
    }

}
```

你在访问`/hello?word=final`时，得到的结果：

```json
{
  "status": 0,
  "description": "success",
  "code": "0",
  "message": "success",
  "data": "hello final!",
  "trace": "7aba435f-69d2-4c44-a944-315107623a92",
  "timestamp": 1605063263491,
  "duration": 0.063,
  "address": "127.0.0.1:80",
  "locale": "en",
  "timeZone": "Asia/Shanghai",
  "success": true
}
```

## 全局异常处理

你在系统中抛出的异常：

```java

@RestController
@RequestMapping("/hello")
public class HelloController {

    @RequestMapping("/ex")
    public void ex(Integer code, String message) {
        throw new ServiceException(code, message);
    }

}
```

你在访问`/hello/ex?code=123&message=exception`时，得到的结果：

```json
{
  "status": 123,
  "description": "exception",
  "code": "123",
  "message": "exception",
```  
