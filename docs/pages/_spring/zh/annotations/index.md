# @Annotation

Spring 提供了大量的`@Annotation`以提高开发效率。

如：

* 用于包扫描的`@Component`及其扩展`@Repository`、`@Service`、`@Controller`、`@RestController`，`@Configuration`等；
* 用于简化事件监听`ApplicationListener`的`@EventListener`。
* 用于Bean注入的`@Bean`。
